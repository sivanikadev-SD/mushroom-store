Dark Mode & RTL

Dark Mode

Activate by toggling the data-theme attribute on <html>. A toggle button is included in every page header.

document.documentElement.setAttribute('data-theme', 'dark');

RTL

Activate by setting the dir attribute on <html>. A toggle button is available in every page header.

document.documentElement.dir = 'rtl';
